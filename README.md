# chat_demo
## install
### ソースコードのクローン
```
git clone git@github.com:atofuzi/chat-demo.git
```
### コンテナビルド・起動 & パッケージインストール
```
make install
```

### LaravelアプリケーションKey生成
```
make key
```
### DBテーブル生成
```
make migrate
```

### DBデータ生成
```
make dbseed
```

## Authentication
Laravel/passportインストール済み
